# Сельскохозяйственный симулятор
Симулятор представляет собой поле из клеток. Каждая клетка изначально сухая, может быть увлажнена ведром.
Есть возможность садить растения, которые могут жить только при определённом уровне влажности.
Среди растений и их предельной влажности: кактус (0-30%), картошка (30-70%), болотник (70-100%).
Основные инструменты: лопата (позволяет выкапывать посаженные растения) и ведро (позволяет увлажнять клетку).
Состояние растений меняется в реальном времени. К примеру, если посадить картошку в сухую клетку, то она умрёт. Но если после этого добавить рядом с ней достаточно влаги, то она будет жить.
